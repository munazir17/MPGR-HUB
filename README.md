The Home of MoneyPaiger Ecosystem — Next.js 15, TypeScript, Tailwind, wagmi, RainbowKit, Base Mainnet.

## Phase 1 (Current)
Project scaffold, wallet connect (RainbowKit + Base), dashboard shell with placeholder stats.

## Setup
1. Get a WalletConnect Project ID at https://cloud.reown.com
2. Copy `.env.example` to `.env.local` and fill it in
3. `npm install`
4. `npm run dev`

## Deploy to Vercel
1. Push this repo to GitHub
2. Import it in Vercel
3. Add the env vars from `.env.example` in Vercel project settings
4. Deploy
